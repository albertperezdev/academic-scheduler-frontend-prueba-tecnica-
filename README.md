# Frontend – Horario Académico (Angular 20)

Este es el frontend del sistema **Gestor de Horarios Académicos**, desarrollado con Angular 20, PrimeNG, TailwindCSS y NgRx.  
Consume un backend construido en NestJS.

---

## Tecnologías

- Angular 20
- TailwindCSS
- PrimeNG
- NgRx (Entity, Effects, Store)
- Signals

---

## Instalación

Clonar el repositorio:

```bash
git clone <url-del-repo>
cd frontend
```

Instalar dependencias:

```bash
npm install
```

---

## ▶️ Ejecutar la aplicación

```bash
ng serve -o
```

Se abrirá automáticamente en:

**http://localhost:4200**
